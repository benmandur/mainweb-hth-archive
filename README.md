# mainweb-hth-archive

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容不针对任何特定域名或具体网站，而是作为一个通用存档库，用于存放各类静态 HTML 资源。仓库结构清晰，便于浏览和引用。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（如图片、样式、脚本）
├── index.html      # 仓库首页（可选）
└── README.md       # 本说明文档
```

- **pages/**：每个 HTML 页面独立存放，命名规则见“页面归档说明”。
- **assets/**：按类型或页面组织，确保外部资源可被正确引用。

## 页面归档说明

- 所有 HTML 页面应保持独立，尽量减少对外部资源的依赖。
- 如需引用外部资源，请将其放置在 `assets/` 目录下，并使用相对路径。
- 页面命名建议使用英文或拼音，避免特殊字符，以保持 URL 友好。
- 每个页面可附带简要说明（可选），记录在 pages 目录下的同名 `.md` 文件中。

## 维护说明

- 本仓库由维护者不定期更新，添加新的 HTML 页面或修正已有内容。
- 欢迎提交 Issue 或 Pull Request，但请注意：内容需符合通用存档定位，不针对任何具体平台或服务。
- 提交前请确保 HTML 文件格式正确，无外部失效链接。
- 不建议在此仓库中存放敏感信息或动态内容。

## 使用方式

- 直接通过 GitHub Pages 或其他静态托管服务访问仓库中的 HTML 文件。
- 也可克隆到本地，使用浏览器直接打开页面文件。
